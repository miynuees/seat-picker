오늘의 자리뽑기

교실에서 바로 쓰는 자리 배정 도구

📌 사용 매뉴얼

🌸 이게 뭐예요?

학기 초·매주 월요일·체험학습 후 — 자리를 다시 정해야 하는 그 모든 순간에 쓰는 도구예요. 학생 명단을 넣고 교실 모양을 잡은 다음, 버튼 하나로 자리를 뽑아요.

4가지 애니메이션 연출이 있어서 학생들이 두근두근 기다려요. 결과는 이미지로 저장해서 학급방에 그대로 올릴 수 있어요.

📱
PWA · 오프라인
한 번 접속하면 인터넷 없이도 동작해요.
💾
자동 저장
명단·교실 설정·테마가 그대로 유지돼요.
🔒
비밀 조건
고정 자리·떨어뜨리기를 학생 몰래 반영.
📷
이미지 저장
결과를 PNG로 받아 그대로 공유.

🚀 4단계로 끝내기

  1. 학생 명단 입력 엑셀 파일을 드래그하거나 직접 한 명씩 입력해요.
  2. 교실 모양 설정 1인·짝꿍·4인 모둠·6인 모둠 중에서 골라요. 가로/세로 칸 수도 조절해요.
  3. (선택) 비밀 조건 설정 특정 학생을 특정 자리에 고정하거나, 떨어뜨려야 하는 학생들을 지정해요.
  4. 자리뽑기 → 결과 저장 4가지 연출 중 하나로 자리를 뽑고, 결과를 이미지로 저장해요.

📝 학생 명단 입력

방법 1 — 엑셀 파일 올리기

업로드 영역에 엑셀 파일(.xlsx, .xls)을 끌어다 놓거나 클릭해서 골라요.

📋 양식 — 첫 번째 열에 번호, 두 번째 열에 이름이 있으면 돼요. 헤더 행이 있어도 자동으로 무시돼요.

방법 2 — 직접 입력

학번·이름을 입력하고 Enter를 누르면 추가돼요. 이름 옆 ✕ 버튼으로 삭제할 수 있어요.

👫 학급 유형 — 단성 학교 / 남녀공학

학급 이름 옆에서 학급 유형을 고를 수 있어요. 단성 학교(여학교·남학교)면 기존처럼 똑같이 쓰시면 돼요. 남녀공학을 고르시면 성별 입력 UI가 나타나요.

📋 공학 모드에서 성별 입력 방법

  • 한 명씩 추가 — 이름 입력란 옆 ♂ 남 / ♀ 여 토글을 눌러 지정한 뒤 추가해요.
  • 붙여넣기·나이스 엑셀 — 학생을 먼저 불러온 뒤, 아래 학생 카드의 동그란 뱃지(? → ♂ → ♀)를 클릭해서 한 명씩 지정해요. 빨간 점선으로 깜빡이는 ? 뱃지는 아직 성별이 안 정해진 학생이에요.
  • 일괄 지정 — 학생 목록 위의 전체 남자로 / 전체 여자로 버튼으로 먼저 일괄 설정한 뒤, 예외인 학생만 개별로 바꾸면 편해요.

⚠️ 주의할 점

  • 번호가 중복되면 나중에 입력한 학생이 덮어써져요.
  • 명단을 다 지우면 자리뽑기를 시작할 수 없어요.
  • 남녀공학 모드에서 성별이 지정되지 않은 학생이 있으면 자리뽑기를 시작할 수 없어요.

🏫 교실 모양 만들기

우리 반 책상 배치 그대로 만들어요. 5가지 템플릿 중에 골라서 시작해요.

🪑
1인 책상
한 명씩 떨어져 앉는 기본 배치
👫
짝꿍 (2인)
둘이 짝지어 앉는 흔한 배치
🧑‍🤝‍🧑
4인 모둠
모둠 활동에 좋은 ㅁ자 배치
👥
6인 모둠
큰 책상·6명 모둠에 맞춤
🎨
커스텀 모둠
가로·세로 직접 지정 (3인 줄, 8인 모둠 등)

가로/세로 칸 조절 + 비활성 자리

슬라이더로 가로 칸·세로 줄 수를 조절해요. 자리를 클릭하면 사용 안 함으로 바뀌어요. 고장난 책상이나 비워둘 자리에 쓰면 좋아요.

🎨 커스텀 모둠 — 자유롭게 모둠 구성

커스텀 모둠을 고르시면 모둠 한 칸의 가로·세로 인원을 직접 정할 수 있어요. 예를 들어 3×1 = 3인 줄, 2×3 = 6인 세로 모둠, 4×2 = 8인 모둠처럼 원하는 대로 구성할 수 있어요. 모둠 결과 화면에서는 같은 모둠 자리들이 연한 핑크 박스로 묶여서 한눈에 보여요.

👫 남녀공학 — 짝꿍 규칙 & 자리별 성별 고정

공학 모드에서는 교실 모양 아래에 두 가지 성별 조절 도구가 나타나요.

💑 짝꿍 규칙 (짝꿍 2인 배치일 때만)

  • 상관 없음 — 성별 무관 (기본)
  • 남녀 짝꿍 — 짝은 남녀 쌍으로 맞춤. 남녀 수가 안 맞으면 남은 쪽은 같은 성별로 채워져요.
  • 같은 성별 짝꿍 — 남남·여여로만 짝을 구성해요.

♂♀ 자리별 성별 고정 (특수 상황용)

미리보기 위의 편집 모드 토글에서 ♂♀ 성별 자리 고정을 선택하면, 자리를 클릭할 때마다 자유 → 남 → 여 → 자유 순서로 바뀌어요. 교탁 앞·창가 등 특정 자리에 특정 성별을 앉혀야 할 때 쓰시면 돼요. 일반 상황에선 쓰지 않으셔도 괜찮아요.

💡 시점 토글 — 결과 화면에서 학생 시점(교탁이 위)과 선생님 시점(교탁이 아래)을 바꿀 수 있어요. 칠판에 띄울 때 편한 쪽으로!

🔒 비밀 조건 (학급 운영 노하우)

학생들에게는 안 보이지만, 자리 배치에는 자연스럽게 반영되는 마법 같은 기능이에요.

  1. 📌 고정 자리 특정 학생을 특정 자리에 미리 배정해요. 시력·키·청력 등 사정이 있는 학생을 앞쪽에 둘 때 써요.
  2. 🚫 떨어뜨리기 같이 앉으면 안 되는 학생 쌍을 지정해요. 짝꿍/모둠에서 자동으로 분리돼요.
  3. 🙅 제외 자리 특정 자리를 자리뽑기에서 제외해요. 책상이 부족하거나 구석 자리를 비워둘 때 OK.

⚠️ 조건이 너무 많으면

  • 모든 조건을 만족하는 배치가 불가능할 수 있어요.
  • 그때는 알림이 뜨고, 조건 일부를 풀어줘야 해요.

🎲 자리뽑기 — 4가지 연출

학생들이 두근두근하는 그 순간! 분위기에 맞춰 골라 쓰면 돼요.

🎰

🃏

덮인 카드 한 장이 짠~ 하고 뒤집혀요.

모든 자리가 한꺼번에 채워져요. 시간 없을 때.

속도 조절

슬라이더로 한 명당 뽑는 속도를 바꿀 수 있어요. 느림은 1교시 분위기 띄우기, 빠름은 종 치기 직전에 추천!

💡 다시 뽑기 — 진행 중에도 다시 뽑기 버튼을 누르면 처음부터 새로 뽑혀요. 애니메이션은 자동으로 취소돼요.

🎉 결과 저장하기

자리 배치가 끝나면 한 장의 이미지로 깔끔하게 남겨요.

  1. 📷 이미지로 저장 버튼을 누르면 PNG 파일로 다운로드돼요. 학급방·SNS·인쇄 어디든 그대로!
  2. 👀 시점 토글 학생 시점(교탁 위쪽) ↔ 선생님 시점(교탁 아래쪽) 전환. 칠판 띄우기에 편한 쪽으로.
  3. ↺ 처음으로 결과를 확인했으면 처음으로 돌아가요. 명단·교실 설정은 그대로 유지돼요.
💡 인쇄가 필요하면 — 📷 이미지로 저장 버튼으로 PNG를 받은 뒤, 일반 사진처럼 인쇄하세요. 가장 깨끗하게 나와요.

📲 앱처럼 설치하기

이 도구는 PWA예요. 한 번 설치하면 인터넷 없이도 쓸 수 있고, 바탕화면에서 바로 열 수 있어요.

💻 PC (Chrome / Edge)

  1. 주소창 오른쪽 끝의 ⊕ 아이콘 클릭 "앱 설치" 또는 "설치" 버튼이 떠요.
  2. "설치" 누르기 바탕화면·시작메뉴에 자리뽑기 앱이 생겨요.
  3. 다음부터는 앱 아이콘으로! 브라우저 안 거치고 바로 열려요. 와이파이가 끊겨도 동작해요.

❓ 자주 묻는 질문

입력한 명단·설정이 다음에도 남아있나요?
네, 같은 브라우저에서 열면 그대로 이어서 써요. 브라우저 데이터를 지우거나 다른 기기로 옮기면 사라지니 주의하세요.
학급을 여러 개 관리할 수 있나요?
현재는 한 번에 한 학급만 저장돼요. 여러 학급을 쓰려면 자리뽑기 결과를 이미지로 저장해둔 뒤, 새 학급 명단으로 갈아끼우면 돼요.
비밀 조건이 학생들에게 보이나요?
전혀 보이지 않아요. 결과 화면에는 자리만 나오고, 어떤 조건이 적용됐는지 표시되지 않아요. 안심하고 쓰세요!
같은 자리뽑기를 여러 번 보여줄 수 있나요?
네! 결과 화면에서 ↺ 처음으로 버튼을 눌러 자리뽑기 단계로 돌아가서 다시 실행하면 매번 새로 뽑혀요.
모바일에서도 잘 동작하나요?
스마트폰·태블릿에서도 동작해요. 다만 칠판이나 TV에 띄울 때는 PC가 화면이 커서 더 보기 좋아요.
결과 화면 글씨가 너무 크거나 작아요.
교실 칸 수가 많으면 자동으로 줄어들고, 적으면 커져요. 화면 크기에 맞게 자동 조정되니 그대로 쓰면 OK!
